# Features and Pages

This covers all the visual and functional features of Glance. Starting from button presses to loading new pages. Basically, what the app does and delivers. 

## Installation

During installation, the app would have a custom installation window that is inlined with the design and themes of the app. The electron installer would be stripped down from the ground up, making it more simpler and modern.

The installation process should be as straight forward as ever: 
1.  Click on the **glance_setup.exe**.
2.  A small greeting for the user. There should be a button labelled **Terms & Agreements** which would get you to the Terms & Agreements Page. And a **Next** or **Continue** button to proceed to the next process.
3.  After the welcome screen, the user should be able to select where to install the app on their drive.
4.  After selecting the installation path, the app should now be ready to install and can be initiated by clicking the **Install** button.
5.  After a successful installation, the installation window will show a **Installation was a Success!** message and a button labelled **Close** to exit the installer. There should also be a checkbox whether to start the app after closing the installer or not.

>   Fun illustrations are encouraged to make the installer and the app more appealing and modern. The installer window should relatively close to the actual app window if possible.

## Launching the App

After a successful installtion, the can now be launched by searching **Glance** from your apps folder. Upon starting a splash screen of the Glance app's logo and name as a loading indicator or just startin the app, it should fade in and out of the scene.
